113. Nearly prime numbers

time limit per test: 0.5 sec.
memory limit per test: 4096 KB

Nearly prime number is an integer positive number for which it is possible to find such primes P1 and P2 that given number is equal to P1*P2. There is given a sequence on N integer positive numbers, you are to write a program that prints УYesФ if given number is nearly prime and УNoФ otherwise.

Input

Input file consists of N+1 numbers. First is positive integer N (1£N£10). Next N numbers followed by N. Each number is not greater than 109. All numbers separated by whitespace(s).

Output

Write a line in output file for each number of given sequence. Write УYesФ in it if given number is nearly prime and УNoФ in other case.

Sample Input

1
6

Sample Output

Yes
